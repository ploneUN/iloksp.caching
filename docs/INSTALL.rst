iloksp.caching Installation
---------------------------

To install iloksp.caching using zc.buildout and the plone.recipe.zope2instance
recipe to manage your project, you can do this:

* Add ``iloksp.caching`` to the list of eggs to install, e.g.:

    [buildout]
    ...
    eggs =
        ...
        iloksp.caching

* Re-run buildout, e.g. with:

    $ ./bin/buildout
